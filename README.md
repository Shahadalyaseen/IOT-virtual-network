# IoT Virtual Network

A secure web-based IoT Virtual Network Management System built using PHP and MySQL.

## Features

* Secure User Authentication
* User Registration
* Password Hashing (bcrypt)
* Session Management
* Access Control
* IoT Device Management
* Alerts Monitoring
* Administrative Dashboard

## Security Features

* Secure password storage using bcrypt hashing
* Session-based authentication
* User validation and access control
* Protected system resources

## Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* XAMPP

## Installation

1. Download the project files.
2. Place the project folder inside the XAMPP `htdocs` directory.
3. Start Apache and MySQL.
4. Import the `db.sql` database file.
5. Configure `config.php`.
6. Open the project in your browser.
